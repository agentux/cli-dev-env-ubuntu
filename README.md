# cli-dev-env-ubuntu
An opinionated preinstalled cli env with zsh, neovim, nodejs, git and more

## Install with git
```shell
git clone git@github.com:zeachco/cli-dev-env-ubuntu.git ~/.cli-dev && bash ~/.cli-dev/setup.sh
```
